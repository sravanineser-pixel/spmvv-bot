# SATHI - SPMVV Doubts Bot 💜

**Your SPMVV Guide** - A chatbot to help students with hostel, course, and admission queries for Sri Padmavati Mahila Visvavidyalayam.

### 🌐 Live App
**Link**: https://sathi-spmvv-bot-8ruvx2vmfazxraau77epyr.streamlit.app/

### ✨ Features
- **Hostel Info**: Names, fees ~₹4,000/month, admission process
- **Courses**: B.Tech CSE/ECE/MECH/EEE, B.Sc Nursing, B.Pharm, MBA, MCA, M.Tech, M.Sc, PhD
- **Admissions**: Complete details for AP EAPCET, AP ICET, AP PGCET, AP LAWCET
- **About SPMVV**: Founded 1983 by Sri N.T. Rama Rao, NAAC A+ Grade, NIRF 2025 Rank 60th in Pharmacy
- **Fests**: College Day, Dept Fests, Hostel Day

### 🛠️ Tech Stack
- **Framework**: Streamlit
- **Language**: Python
- **Hosting**: Streamlit Community Cloud
- **Version**: 1.0
